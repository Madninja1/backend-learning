# Day 17 — PHP Internals (Autoload & Class Loading)

file: php-internals-lab/day17-autoload.md

## Тема

Как PHP автоматически загружает классы (autoload)

---

## 1. Проблема без autoload

Раньше нужно было писать:

```php
require "User.php";
require "Order.php";
require "Product.php";
```

Проблемы:

* много require
* сложно поддерживать
* легко забыть подключить файл

---

## 2. Что такое autoload

Autoload — это механизм, который автоматически загружает классы.

Когда PHP видит:

```php
$user = new User();
```

Если класс не найден:

→ вызывается autoload

---

## 3. spl_autoload_register

Основной механизм:

```php
spl_autoload_register(function ($class) {
    require $class . ".php";
});
```

---

## Как это работает

1. PHP не находит класс
2. вызывает autoload функцию
3. она подключает файл
4. класс становится доступен

---

## 4. Пример

```php
spl_autoload_register(function ($class) {
    require "classes/" . $class . ".php";
});

$user = new User();
```

PHP:

* вызывает autoload
* подключает classes/User.php

---

## 5. Несколько autoload функций

Можно зарегистрировать несколько:

```php
spl_autoload_register($fn1);
spl_autoload_register($fn2);
```

PHP будет вызывать их по очереди.

---

## 6. Composer autoload

В реальности используется Composer:

* генерирует autoload
* работает через PSR-4

---

## 7. PSR-4 (упрощённо)

Namespace → путь к файлу

Пример:

```php
App\Services\UserService
```

→

```text
/app/Services/UserService.php
```

---

## 8. Почему это важно

Autoload:

* убирает require
* делает код чище
* основа современных фреймворков

---

## 9. Производительность

Autoload:

* вызывается только при необходимости
* OPcache ускоряет загрузку файлов

---

## 10. Ошибки

Если файл не найден:

→ fatal error (класс не найден)

Это механизм автоматической загрузки классов.

Когда PHP не находит класс при использовании.

Ответ:

Регистрирует функцию, которая будет загружать классы.


Если класса нет → вызывается autoload → подключается файл

Да, PHP вызывает их по очереди.

Стандарт сопоставления namespace → файл.

# Мини-итог

Autoload позволяет автоматически загружать классы без ручных require.
Он вызывается при первом использовании класса и подгружает нужный файл.
Это основа современной архитектуры PHP-приложений и работы Composer.
